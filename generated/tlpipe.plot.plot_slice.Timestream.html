tlpipe.plot.plot_slice.Timestream¶

class tlpipe.plot.plot_slice.Timestream(files=None, mode='r', start=0, stop=None, dist_axis=0, use_hints=True, comm=None)[source]¶

Container class for the timestream data.

This timestream data container is to hold time stream data that has polarization and baseline separated from the channel pairs in the raw timestream.

:param Same as container.BasicTod.:
